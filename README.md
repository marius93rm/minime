# minime - Starter Structure for Codex

Struttura pronta per avviare un progetto con Codex, con agenti specialistici e template operativi.

## Struttura principale

- `.codex/` configurazione agenti e template.
- `src/` codice applicativo (backend, frontend, shared).
- `tests/` struttura test (unit, integration, e2e).
- `docs/` documentazione tecnica e operativa.
- `infra/` risorse infrastrutturali e CI/CD.
- `scripts/` automazioni.

## Uso rapido

1. Apri `.codex/agents/00-orchestrator.md`.
2. Definisci roadmap e milestone.
3. Esegui gli agenti specialistici in base alla fase.
4. Documenta output in `docs/` usando i template in `.codex/templates/`.

# Codex Workspace

Questa cartella contiene la configurazione operativa per lavorare con Codex in modo strutturato:

- `agents/`: agenti specializzati per ogni fase del progetto.
- `templates/`: template pronti per PRD, ADR, task e checklist.

Flusso suggerito:
1. Parti da `agents/00-orchestrator.md`.
2. Usa gli agenti specialistici in sequenza in base alla fase.
3. Registra decisioni e output in `docs/` usando i template.
